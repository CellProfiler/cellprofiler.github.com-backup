CellProfiler Analyst examples

Example data and a properties file for CellProfiler Analyst can be downloaded here. This is the easiest way to get up and running with the software, and will help you to understand which fields are required in the properties file to get up and running.

Example database, properties file, and 384 human cellular images. Also includes an example Classifier training set. This data is also available for download via SVN here.
For use with CellProfiler Analyst 2.0 only.
