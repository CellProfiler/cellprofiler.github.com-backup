How to install CellProfiler on Linux

CentOS 6 and similar distributions

  1. As root, create a file called /etc/yum.repos.d/cellprofiler.repo with the following contents: 
    [cellprofiler]
name=CellProfiler for CentOS 6
baseurl=http://www.cellprofiler.org/linux/centos6/
enabled=1
gpgcheck=0
  2. As root: yum install cellprofiler
  3. As a regular user, type cellprofiler to start CellProfiler. If the DISPLAY environment variable is set, CellProfiler will run in GUI mode.
