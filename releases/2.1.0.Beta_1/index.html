CellProfiler 2.1.0.Beta_1 release

These are the Windows install packages and OS/X DMG for the CellProfiler 2.1.0 Beta 1 release. 2.1.0 Beta 1 is a preview of our upcoming 2.1.0 formal release. It has passed unit tests and is being used by the Imaging Platform for image analaysis, but has not undergone the extensive functionality review of our formal releases. We plan to make some minor changes to 2.1.0 before release. We would appreciate feedback and bug reports. Suggestions should be posted and discussed on cellprofiler-dev and bugs should be reported to our GitHub repository.
CellProfiler's user interface and internals have changed substantially. We'll release documentation and tutorials geared to users upgrading from 2.0 to 2.1 as part of the official release.
