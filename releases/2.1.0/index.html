CellProfiler 2.1.0. release candidate

These are the Windows install packages and OS/X DMG for the CellProfiler 2.1.0 release candidate. We are tentatively planning to release on Jan. 24, 2014. We would appreciate feedback and bug reports. Bugs should be reported to our GitHub repository. We will be triaging the bugs - we've already identified some issues which won't be fixed in this release and it may be that something you discover will also not make it into the release. We are planning to have a follow-on maintenance release in about two months which would include fixes to bugs not addressed in this release.
CellProfiler's user interface and internals have changed substantially. We'll release documentation and tutorials geared to users upgrading from 2.0 to 2.1 as part of the official release.
